Folder eco — это легковесный и минималистичный файловый менеджер для Windows, созданный для максимально быстрой навигации и работы с кодом. Никаких перегруженных интерфейсов — только то, что нужно для продуктивности.


✨ Основные фишки

    🚀 Динамические диски: Быстрое переключение между дисками кнопками. Приложение запоминает ваш последний путь на каждом диске — возвращайтесь ровно туда, где закончили.

    🧩 Умные хлебные крошки: Наглядная навигация по пути с возможностью мгновенного прыжка в любую родительскую папку.

    💻 Интеграция с инструментами: Выделенные кнопки для запуска Терминала и VS Code прямо в текущей папке.

        Лайфхак: Нажми ПКМ по кнопке Терминала или VS Code, чтобы выбрать свой любимый .exe (например, Commander или Sublime).

    🏠 Система «Домашней папки»: Используй обычные Windows-ярлыки для создания списка избранного. Нажми ПКМ на любой папке -> «Отправить ярлык в Домашнюю», и она всегда будет под рукой.

    🟪 Плавающий виджет: Маленький фиолетовый квадрат, который всегда под рукой.

        Одинарный клик: Свернуть/Развернуть менеджер.

        Двойной клик: Переметнуть виджет на другой край экрана.

        Перетаскивание: Поставь его там, где удобно.

    📥 Трей-режим: Приложение не мешается на панели задач, сворачиваясь в системный трей.

🛠 Технологии

    Язык: C#

    Платформа: .NET / WinForms

    Стиль: Минимализм и высокая скорость работы.
